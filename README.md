# 판다마켓

- 배포주소 : https://panda-market-kr.netlify.app/

이 프로젝트는 주로 HTML, CSS, JavaScript로 작성된 웹 애플리케이션입니다. 

- `index.html`: 판다마켓의 메인 페이지로, 사용자가 상품을 거래할 수 있는 플랫폼을 소개합니다. 주요 섹션으로는 히어로 섹션, 인기 상품, 검색 기능, 상품 등록 기능 등이 포함되어 있습니다.
- `login/index.html`: 판다마켓의 로그인 페이지로, 사용자가 이메일과 비밀번호를 입력하여 로그인할 수 있는 폼을 제공합니다. 또한, 구글과 카카오를 통한 간편 로그인 기능도 포함되어 있습니다.
- `signup/index.html`: 판다마켓의 회원가입 페이지로, 사용자가 이메일, 닉네임, 비밀번호를 입력하여 회원가입할 수 있는 폼을 제공합니다. 비밀번호 확인 기능도 포함되어 있습니다.
- `css/login.css`: 로그인 페이지의 스타일을 정의한 CSS 파일입니다. 로그인 폼, 간편 로그인, 회원가입 링크 등의 스타일이 포함되어 있습니다.
- `css/common.css`: 공통 스타일을 정의한 CSS 파일로, 다양한 컴포넌트의 스타일을 포함하고 있습니다. 

이 프로젝트는 판다마켓이라는 중고거래 플랫폼을 위한 웹 애플리케이션으로, 사용자에게 안전하고 신뢰할 수 있는 거래 환경을 제공합니다.
